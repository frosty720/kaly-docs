# Wallet Setup

KalyChain supports any EVM-compatible wallet. The most popular choice is **MetaMask**.

## Connecting MetaMask

You can add KalyChain to MetaMask manually using the network details below.

### 1. Open Network Settings

Click the network selector at the top of your MetaMask extension and choose **"Add Network"**.

### 2. Enter Network Details

#### For Mainnet

- **Network Name**: KalyChain Mainnet
- **New RPC URL**: `https://rpc.kalychain.io`
- **Chain ID**: `3888`
- **Currency Symbol**: `KLC`
- **Block Explorer URL**: `https://kalychain.io`

#### For Testnet

- **Network Name**: KalyChain Testnet
- **New RPC URL**: `https://testnetrpc.kalychain.io/rpc`
- **Chain ID**: `3889`
- **Currency Symbol**: `KLC`
- **Block Explorer URL**: `https://testnet.kalychain.io`

### 3. Save

Click **Save**. You should now see KLC as the native token and be able to interact with dApps on KalyChain.

## Getting Testnet Funds

To start building on the testnet, you can acquire testnet KLC from the official faucet:

> **[KalyChain Testnet Faucet](https://faucet.kalychain.io)**

Enter your wallet address to receive test KLC for gas fees.
