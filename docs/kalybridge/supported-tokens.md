# Supported Tokens

KalyBridge supports bridging the following assets between KalyChain and BSC.

## Token List & Contract Addresses

To view bridged tokens in your wallet on **KalyChain**, import the following contract addresses.

| Token | Symbol | Contract Address (KalyChain) |
|-------|--------|------------------------------|
| **Native KLC** | KLC | Native (Gas Token) |
| **Wrapped BNB** | BNB | `0x0e2318b62a096AC68ad2D7F37592CBf0cA9c4Ddb` |
| **Tether** | USDT | `0x2CA775C77B922A51FcF3097F52bFFdbc0250D99A` |
| **DAI** | DAI | `0x6E92CAC380F7A7B86f4163fad0df2F277B16Edc6` |

## How to Add Tokens to MetaMask

1. Open MetaMask.
2. Ensure you are on the **KalyChain** network.
3. Click **"Import tokens"** (or "Add Token").
4. Paste the **Contract Address** from the table above.
5. The symbol and decimals should auto-fill.
6. Click **Add Custom Token**.
