# KalyBridge Overview

**KalyBridge** uses Hyperlane Warp Routes to make it safe and easy to bridge your tokens to and from **KalyChain** and **Binance Smart Chain (BSC)**.

## Key Features

- **Secure**: Powered by Hyperlane's interchain messaging protocol.
- **Fast**: Quick transaction finality between BSC and KalyChain.
- **User Friendly**: Simple interface for moving assets cross-chain.

## Supported Networks

- **From**: Binance Smart Chain (BSC)
- **To**: KalyChain
- *(and vice versa)*
