# Supported Tokens

KalyBridge supports bridging the following assets between KalyChain and BSC.

## Where to Bridge

- **[KalyBridge](https://bridge.kalychain.io/)** — Dedicated bridge interface
- **[KalySwap Bridge](https://app.kalyswap.io/bridge)** — Integrated into KalySwap DEX

## Token List & Contract Addresses

Click "Add to MetaMask" to quickly add any token to your wallet.

## Manual Token Import

If the "Add to MetaMask" button doesn't work, add tokens manually:

1. Open MetaMask
2. Ensure you're on the **KalyChain** network
3. Click **"Import tokens"**
4. Paste the contract address
5. Symbol and decimals should auto-fill
6. Click **Add Custom Token**
